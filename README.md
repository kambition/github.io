# 🫀 人体解剖探索 - 3D交互式生物学习系统

一个专为初中生物教学设计的3D交互式人体解剖学习系统，让学生能够身临其境地探索人体器官和循环系统的奥秘。

## ✨ 主要功能

### 1. 完整的人体系统
- **🦴 骨骼系统**：白色骨架结构，包括颅骨、脊柱、肋骨、骨盆等
- **🫀 心脏**：四腔室结构，实时搏动动画
- **🫁 肺部**：左右肺叶，呼吸动画效果
- **🟫 肝脏**：人体最大的内脏器官
- **🟣 胃**：J形结构的消化器官

### 2. 血液循环系统
- **🔴 动脉系统**：主动脉、肺动脉及分支
- **🔵 静脉系统**：上下腔静脉、肺静脉
- **💜 毛细血管网络**：遍布各个器官的精细血管网

### 3. 循环动画（循环播放）
- **体循环（大循环）**：左心室 → 主动脉 → 全身 → 静脉 → 右心房
- **肺循环（小循环）**：右心室 → 肺动脉 → 肺部 → 肺静脉 → 左心房
- **实时血液流动**：250+血液粒子实时循环流动
- **颜色变化**：肺循环中血液从蓝色（缺氧）渐变为红色（含氧）

### 4. 交互功能
- **点击探索**：点击任何器官查看详细介绍
- **快速导航**：7个快捷按钮切换不同系统
- **360°旋转**：鼠标拖动自由旋转视角
- **缩放功能**：滚轮缩放查看细节
- **智能聚焦**：自动聚焦到选中的器官

### 5. 教育内容
每个器官都包含初中生物课程的知识点：
- 器官的结构特点
- 主要功能
- 科学知识
- 趣味数据

### 快捷按钮
- **骨骼系统**：查看人体支架结构
- **心脏**：了解生命的泵
- **肺**：气体交换站
- **肝脏**：化学工厂
- **胃**：食物加工厂

- **重置视图**：返回初始视角

## 📖 教学建议

### 课堂使用
1. **导入环节**：全景展示激发学生兴趣
2. **讲解环节**：点击相应器官配合讲解
3. **探索环节**：让学生自由探索
4. **复习环节**：通过可视化加深记忆

### 自主学习
- 学生可以按照自己的节奏探索
- 通过点击不同部位主动学习
- 观察血液循环过程理解生命活动
- 多角度观察器官结构

## 🎯 设计理念

- **直观性**：3D可视化比平面图更直观
- **交互性**：主动探索比被动接受更有效
- **美观性**：精美的视觉效果提升学习兴趣
- **科学性**：严格遵循生物学知识
- **适龄性**：内容难度适合初中生

## 🌟 特色亮点

1. **首创白色骨骼 + 红色器官**的对比设计
2. **循环播放的血液流动动画**，永不停歇
3. **颜色渐变**展示肺部气体交换过程
4. **250+粒子**模拟真实血液流动
5. **7个完整系统**可供探索
6. **物理级渲染**带来电影般的视觉体验
