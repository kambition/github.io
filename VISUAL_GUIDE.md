## ⚡ 动画参数

### 心脏搏动
```javascript
频率：3 Hz (180 BPM - 加速显示)
收缩期：scale = 1 + sin²(t*3) * 0.1
舒张期：scale = 1 - |sin(t*3)| * 0.06
旋转：y轴 ±3.4°, x轴 ±1.7°
```

### 血管弹性
```javascript
动脉（强弹性）：
  脉动 = sin(t * 3) * 0.15 + 1
  范围：85% - 115%
  
静脉（弱弹性）：
  脉动 = sin(t * 2) * 0.05 + 1
  范围：95% - 105%
  
毛细血管（无弹性）：
  脉动 = 1
  范围：100% 恒定
```

### 红细胞运动
```javascript
位置：progress += 0.0003/帧
旋转X：+0.02 * rotationSpeed
旋转Y：+0.015 * rotationSpeed
缩放：squeeze = sin(t*2 + progress*10) * 0.1 + 1
形变：scale(squeeze, 1/squeeze, squeeze)
```

## 🔬 教学观察要点

### 1. 血流速度对比
```
观察粒子移动速度：
🔴 动脉粒子：快速移动，明显脉动
🔵 静脉粒子：中速移动，平稳流动
💜 毛细血管红细胞：极慢移动，单行通过

速度比例 ≈ 40 : 15 : 0.5 (cm/s)
```

### 2. 血管弹性对比
```
观察右侧3D管道模型：
🔴 动脉：强烈搏动 (±15%)
🔵 静脉：轻微搏动 (±5%)
💜 毛细血管：无搏动
```

### 3. 管壁结构对比
```
观察右侧横切面：
🔴 动脉：厚管壁，3层结构
🔵 静脉：薄管壁，2层结构
💜 毛细血管：极薄，1层细胞
```

### 4. 连接网络观察
```
选择任一器官，观察：
动脉分支 (红色) → 毛细血管网 (紫色) → 静脉汇聚 (蓝色)
颜色渐变：红 → 粉红 → 紫 → 紫蓝 → 蓝
```

## 🚀 使用流程建议

### 初学者流程
1. 点击"血液循环" → 观察整体系统
2. 点击"心脏" → 理解心脏作为泵的作用
3. 观察红色粒子从心脏出发 → 动脉系统
4. 观察蓝色粒子回到心脏 → 静脉系统
5. 观察紫色毛细血管中的红细胞慢速通过

### 进阶学习流程
1. 点击"动脉" → 观察树状分支结构
2. 点击"静脉" → 观察汇聚结构
3. 点击"毛细血管" → 观察A→C→V连接
4. 观察右侧对比区 → 理解结构差异
5. 对比粒子速度 → 理解功能差异

### 考试复习流程
1. 观察并记忆两条循环路径
2. 对比三种血管的结构特点
3. 记忆血流速度的巨大差异
4. 特别注意肺动脉/肺静脉的特殊性
5. 理解毛细血管的物质交换功能
