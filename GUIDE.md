# 🎯 快速开始指南

## 欢迎使用人体解剖3D探索系统！

这是一个专为初中生物教学设计的交互式3D学习工具。

## 📺 如何开始

1. **打开预览窗口**
   - 点击界面上的"预览"按钮
   - 或直接访问：http://localhost:5173

2. **等待加载**
   - 首次打开需要加载3D模型
   - 看到"加载中..."消失后即可开始探索

## 🎮 基础操作

### 鼠标控制
```
鼠标左键拖动 = 旋转整个3D场景
鼠标滚轮     = 放大/缩小视图
点击器官     = 查看该器官的详细介绍
```

### 快捷按钮（底部）
```
骨骼系统   = 查看人体骨架结构
心脏       = 聚焦心脏并查看其四腔室结构
肺         = 查看左右肺叶和呼吸系统
肝脏       = 了解人体最大的内脏器官
胃         = 探索消化系统
血液循环   = 观看完整的血液循环动画
重置视图   = 返回初始视角
```

## 🌟 特色功能详解

### 1. 骨骼系统 🦴
- **外观**：白色半透明骨架
- **包含**：颅骨、脊柱、12对肋骨、骨盆
- **作用**：作为其他器官的参考框架
- **操作**：点击"骨骼系统"按钮或直接点击骨骼部分

### 2. 心脏 🫀
- **外观**：红色，有节奏地跳动
- **结构**：左右心房、左右心室
- **动画**：每秒跳动约1.5次
- **血管**：连接主动脉和肺动脉
- **操作**：点击"心脏"按钮聚焦查看

### 3. 肺部 🫁
- **外观**：粉红色，随呼吸起伏
- **结构**：左肺（2叶）、右肺（3叶）
- **动画**：平缓的呼吸运动
- **气管**：连接到气管和支气管
- **操作**：点击"肺"按钮

### 4. 肝脏 🟫
- **外观**：棕色，位于右上腹
- **特点**：人体最大的内脏器官
- **位置**：紧邻心脏下方
- **操作**：点击"肝脏"按钮

### 5. 胃 🟣
- **外观**：粉色，J形结构
- **位置**：左上腹部
- **结构**：胃底、胃体、幽门
- **操作**：点击"胃"按钮

### 6. 血液循环系统 🔄
这是最精彩的部分！

#### 动脉系统（红色）
- 从心脏向外输送血液
- 主动脉是最粗的血管
- 分支到全身各处

#### 静脉系统（蓝色）
- 将血液送回心脏
- 上下腔静脉收集全身血液
- 肺静脉输送含氧血

#### 毛细血管（紫色）
- 最细的血管
- 遍布各个器官周围
- 进行物质交换

#### 血液流动动画 ⭐
**这是本系统的核心特色！**

1. **250+血液粒子**持续循环流动
2. **两条循环路径**同时进行：
   - **体循环（红色粒子）**：左心室 → 全身 → 右心房
   - **肺循环（变色粒子）**：右心室 → 肺 → 左心房

3. **颜色变化**：
   - 肺循环中的血液从蓝色（缺氧）渐变为红色（含氧）
   - 展示了肺部的气体交换过程

4. **永不停歇**：
   - 动画自动循环播放
   - 完美展示"生命在于运动"

**操作**：点击"血液循环"按钮，其他器官会变半透明，血管和血液粒子会高亮显示

## ⚙️ 常见问题

### Q: 画面卡顿怎么办？
A: 
- 关闭其他占用资源的程序
- 使用Chrome或Edge浏览器
- 点击"重置视图"简化场景

### Q: 看不到血液流动？
A: 
- 点击"血液循环"按钮
- 血液粒子会放大显示
- 可能需要等待几秒让粒子分布开

### Q: 如何回到初始视角？
A: 
- 点击"重置视图"按钮
- 所有器官恢复正常显示
- 相机返回初始位置

### Q: 器官介绍文字在哪里？
A: 
- 左上角的白色面板
- 点击器官或按钮后自动更新
- 包含详细的初中生物知识点
